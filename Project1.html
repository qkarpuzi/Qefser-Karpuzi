Eli Proprete - Cleaning Company Website & Admin Dashboard

Eli Proprete is a fully functional web application designed for managing a cleaning service business. The platform includes a user-friendly front-end and a powerful admin dashboard for seamless management. Built with PHP, HTML, CSS, JavaScript, and SQL, the system handles everything from service bookings to user management.

Key Features:

    Admin Dashboard: A robust system for managing bookings, users, services, and other business operations. Database Integration: Secure and efficient data handling using SQL. Full-Stack Development: The entire system, including front-end and back-end, was developed to ensure a smooth user experience and effective business management.

Tools Used:

    PHP, HTML, CSS, JavaScript: For developing a fully functional website. SQL: For secure database management and efficient data storage. Responsive Design: Ensures a smooth experience on both desktop and mobile devices.

This project showcases my expertise in full-stack web development, database management, and system functionality.
