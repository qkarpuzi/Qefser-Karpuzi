Engineering Manual Physio

Manual Physio is a comprehensive clinic management system designed specifically for physiotherapy practices. It manages patient records, appointments, treatment plans, and billing in an integrated platform that streamlines clinic operations.

I developed this system to address the unique needs of physiotherapy clinics — requiring specialized tracking of patient progress, treatment sessions, and rehabilitation plans. The dashboard provides both clinic administrators and healthcare providers with the tools they need to deliver efficient, high‑quality care.

Built with a robust PHP backend (MVC), MySQL database with complex relationships, and a responsive frontend using Bootstrap 5 and custom CSS. Security measures follow HIPAA‑inspired practices (encryption, audit trails, role‑based access).
